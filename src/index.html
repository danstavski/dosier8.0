Технологии этого сайта

VSCode

Visual Studio Code (VSCODE) - текстовый редактор от Microsoft, в котором и создаются множество сайтов. Там используют специальные языки, библиотеки, загружаются изображения и уже со всем этим можно работать

HTML

HTML - язык гипертекстовой разметки, который позвляет делать основу страницы, её структуру (скелет). Весь текст, картинки, ссылки - все HTML. Без него просто невозможно существование любого сайта

CSS (SCSS)

CSS - язык каскадных таблиц стилей, он позволяет делать оформление сайта, воплотить почти любой дизайн. Шрифты, тени, фон, цвета, часть анимации и т.д. делается именно на CSS. SCSS (препроцессор) - продвинутая версия CSS

Gulp

Gulp — это инструмент для автоматизации рутинных задач в веб-разработке. Он упрощает процессы сборки, минификации, конкатенации файлов, оптимизации изображений, запуска тестов и обновления браузера

Bootstrap

Bootstrap - библиотка CSS, с помощью которой можно быстро делать констуркции разной сложности для сайта, она ускоряет разработку и позволяет делать быстро сложные задачи на сайте, которые самому делать долго

JavaScript

JavaScript (JS) - язык програмирования, на котором пишется весь Frontend без исключения. Позволяет взаимодействовать с элементами страниц и работать с сервером сайта. Самая сложная технология в этом списке

CapCut

CapCut - не требовательный видеоредактор, который позволяет редактировать, склеивать, улучшать качество и цветокарекцию видео. С помощью него я сделал два красивых видео-фона на других страницах сайта
